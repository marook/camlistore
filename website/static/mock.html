Perkeep:

  • is a way to store, sync, share, model and back up content
  • is a work in progress
  • is Open Source (Apache licensed)
  • is content-addressable
  • is indexed and searchable
  • has an API
  • syncs from anywhere to anywhere
  • is your "home directory for the web"
  • is pro-JSON (yet aggressively format agnostic)
  • is pro-GnuPG (for signing claims)
  • is pro-paranoia and privacy
  • is ambitious, but ...
  • is simple!
  • is programning language-agnostic (parts and different implementations in Go, Python, Java, Perl, Bash, ... the language doesn't matter.)
  • is neither "Cloud" nor "Local". happily both.
  • is a "20% project" from a few Google employees, but not Google-centric nor endorsed by Google (other than them letting us open source our side project)
  • is not yet ready for users
  • is ready for developers (at least those without strong color preferences)
